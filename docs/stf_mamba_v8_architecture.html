V7.3 → V8.0 Upgrade Path
Phase 1
Spatial Feature Extraction
ConvNeXt V2-Base (ImageNet-22K pretrained, frozen/low-LR). Extracts domain-invariant spatial features — edges, textures, lighting priors — preventing the Transfer Learning Gap.
Input: (B, 3, 32, 224, 224) → Output: (B, 32, 1024)
Phase 2
Frequency Enhancement — W-SS Module
3D-DWT (Symlet-2) decomposes spatial features into 8 wavelet sub-bands. The Wavelet-Selective Scan then processes them through specialized streams instead of flattening.
3D-DWT Output: 8 sub-bands × (B, T/2, H/2, W/2)
⬡ Wavelet-Selective Scan (W-SS) — NEW in V8.0

3 1D Hydra scan (temporal axis only)
4 Output: Ftemporal ∈ ℝB×T/2×d

"Adapts to compression profile per-input"
Ffused = Hydraspatial(SHHH) ⊙ σ(Hydratemporal(SHLL)) Spatial features gated by temporal flicker presence — suppresses false positives from static high-frequency textures
Phase 3
Temporal Modeling — PN-Mamer Blocks ×4 stages
Each stage interleaves N=3 Hydra mixers with 1 Attention layer (MamBo-3-Hydra-N3 config). The PN-SSM splits into parallel paths for cross-dimensional artifact modeling.
Path A — Temporal Recurrence (Hydra)
Quasiseparable matrix mixer. Single-pass native bidirectionality via MQS = shift(SSfwd) + flip(shift(SSbwd(flip))) + D·X. Tracks sequential drift evolution with dconv=7 (±3 frame context).
Path B — Channel Mixing (PN)
Dense projections + gating across frequency channels. Models HLL↔HHH correlations explicitly. Answers: "Does temporal flicker at this frame co-occur with spatial noise anomaly?"
↓ Cross-Dimensional Fusion (learnable gate) ↓
Mamer Layer — Global Spectral Recall
LayerMamer(x) = Attention(Norm(Mamba(Norm(x)) + x)) + (Mamba(x) + x)
Replaces FFN with MHA — enables direct comparison of spectral signatures across all frames
⚡ Adaptive Sdim Scaling (Energy-Gated)
Sdim(V) = dbase × (1 + tanh(γ · EHLL)) — more experts activated for hard samples with high HLL ambiguity

Phase 4
Classification Head
Global Average Pooling → MLP → Softmax. Trained with composite loss.
Ltotal = LCE + λ · LHLL-contrastive Contrastive HLL: max(0, m − (‖HLLfake‖₂ − ‖HLLreal‖₂)) — replaces MSE-to-zero
Output: (B, 2) [Real, Fake]
